# pycalc

**pycalc** est une petite calculatrice en ligne de commande écrite en Python. 
Elle a été créée dans un but pédagogique afin d'illustrer l'utilisation de 
**Git** et la gestion d'un dépôt simple.

Le programme prend trois arguments :
1. une première opérande,
2. un opérateur (`+`, `-`, `*`, `/`),
3. une deuxième opérande.

Il affiche le résultat du calcul ou un message d'erreur si les arguments sont 
invalides.

## Prérequis

- Python 3.x

## Utilisation

Exécuter le script en ligne de commande en fournissant les trois arguments :

```bash
python pycalc.py <opérande1> <opérateur> <opérande2>
```

Par exemple :

- dans Terminal, sur Linux
```bash
python3 pycalc.py 3.5 \* 2.11
```
- dans Invite de commandes ou PowerShell, sur Windows
```bash
python.exe pycalc.py 3.5 * 2.11
```
